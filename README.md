# Требования к проекту "Разработка агента поддержки руководителя команды разработки"

**TeamLead AI Agent** — это аналитическая веб-платформа, предназначенная для автоматизации управленческой рутины, мониторинга состояния команды и оптимизации рабочих процессов в разработке ПО. Система агрегирует данные из ключевых инструментов разработки (Jira, Git, Confluence, Outlook) и предоставляет руководителю единую «точку правды» через интерактивный дашборд и ИИ-ассистента.
