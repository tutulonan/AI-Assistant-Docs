### Jira API: 

**Источники:** Документация по задачам Jira: https://www.atlassian.com/ru/software/jira/guides/issues/overview#what-are-parent-and-child-work%20items

- ID задачи, 
- Сведения:
	- исполнитель,
	- метки,
	- родитель,
	- команда,
	- срок исполнения,
	- дата начала,
	- автор,
	- учёт времени
- тип (Эпик/Задание/История/Баг/Подзадача/Кастомный тип), 
- резюме,
- Основные сведения
	- среда,
	- исходная оценка времени,
	- описание,
- статус, 
- дочерние задачи,
- привязанные задачи и тип связи,
- вложения,
- для каких ролей доступна задача,
- флаг Impediment,
- Issue color, (#priority/low)
- Активность
	- Комментарии,
	- История,
	- Журнал работ,
- история переходов по статусам, 
- сотрудники, отслеживающие задачу,
- голоса.
